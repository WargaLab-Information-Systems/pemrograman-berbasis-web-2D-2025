Halo, nama saya Dhani Kusuma Prasetyo, namun biasa dipanggil Dhani. Saat ini, saya merupakan mahasiswa di Universitas Trunojoyo Madura, dengan mengambil Program Studi S1 Sistem Informasi. Saya berasal dari sebuah desa di Kabupaten Lamongan, tepatnya di Desa Warukulon, Kecamatan Pucuk. Sejak kecil, saya memiliki ketertarikan dalam dunia desain, terutama dalam membuat desain pamflet, stiker, dan typography. Hobi saya dalam mengedit dan menciptakan berbagai desain sudah menjadi bagian dari keseharian saya. Alasan saya memilih Program Studi Sistem Informasi adalah untuk mengembangkan pengetahuan saya lebih dalam lagi, terutama dalam menggabungkan dunia desain dengan teknologi informasi. Saya berharap melalui program studi ini, saya dapat meningkatkan keterampilan saya dan membawa kreativitas dalam bidang yang saya geluti. Terima kasih atas kesempatan ini, dan saya berharap kita bisa saling berbagi pengalaman serta berkembang bersama.
