Puncak Arus Mudik di Daop 8 Surabaya Terjadi Hari Ini, 49.103 Penumpang Gunakan Kereta Api

PT Kereta Api Indonesia (Persero) Daop 8 Surabaya mencatat puncak arus mudik Lebaran 2025 terjadi pada, Jumat (28/3/2025) hari ini. Hingga pukul 09.00 WIB, tercatat sebanyak 49.103 pelanggan menggunakan moda transportasi kereta api, dengan rincian 27.569 pelanggan naik dan 21.534 pelanggan turun di berbagai stasiun wilayah Daop 8.
